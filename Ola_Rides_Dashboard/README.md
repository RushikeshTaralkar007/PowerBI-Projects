
# 🚖 OLA Ride Analytics Dashboard | Power BI

An interactive **Power BI dashboard** designed to analyze OLA ride booking data and provide actionable business insights. The dashboard enables users to monitor bookings, revenue, cancellations, vehicle performance, ride distance, and customer ratings through dynamic and interactive visualizations.

---

## 📌 Project Overview

This Power BI dashboard transforms raw ride booking data into meaningful business insights using interactive reports, DAX measures, and data visualizations.

The dashboard is designed to help stakeholders:

- Monitor overall business performance
- Track ride bookings and revenue
- Analyze vehicle-wise performance
- Understand cancellation trends
- Evaluate customer and driver ratings
