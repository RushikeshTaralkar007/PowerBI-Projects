
# 🍽️ Zomato Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Zomato Sales Dashboard** developed using **Microsoft Power BI**. The dashboard transforms raw food delivery data into meaningful business insights, enabling stakeholders to analyze sales performance, customer behavior, restaurant trends, and order distribution through dynamic visualizations.

The dashboard is designed to support data-driven decision-making by providing an intuitive and interactive reporting experience.

---

## 🎯 Objectives

* Analyze overall sales and order performance.
* Identify top-performing cities and restaurants.
* Compare food category sales.
* Monitor customer ratings and order trends.
* Create an interactive dashboard with slicers and filters.
* Present business insights in a visually appealing format.
