

# 📊 Telecom Customer Churn Analysis | Power BI

## 📌 Project Overview

The **Telecom Customer Churn Analysis Dashboard** is an interactive Power BI project designed to analyze customer behavior, identify churn patterns, and provide actionable business insights. The dashboard enables stakeholders to monitor customer retention, understand churn drivers, and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze customer churn trends
- Identify factors affecting customer retention
- Monitor customer demographics and service usage
- Compare contract types and payment methods
- Generate business insights through interactive dashboards

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel
- Data Modeling

---

## 📂 Dataset

The dataset contains telecom customer information, including:

- Customer Demographics
- Internet Services
- Contract Types
- Payment Methods
- Monthly Charges
- Total Charges
- Customer Tenure
- Churn Status

---

## 📊 Key Insights

- Month-to-month contracts have the highest churn rate.
- Customers using Fiber Optic services show higher churn compared to DSL users.
- Electronic Check is the most common payment method among churned customers.
- Long-tenure customers are more likely to remain with the company.
- Additional services such as Tech Support and Online Security contribute to better customer retention.
